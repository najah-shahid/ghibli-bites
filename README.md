# 🌿 Ghibli Bites

A Studio Ghibli–inspired website built with **React + Vite + TypeScript**, where users can explore iconic foods from Ghibli movies in a playful, animated interface.

- Live Demo: [ghibli-bites.vercel.app](https://ghibli-bites.vercel.app/)

## 🚀 Tech Stack

* ⚛️ **React (Vite + TypeScript)**
* 🎨 **CSS**
* 📦 **Static JSON for data**


## 🌐 Pages & Features

### 🏠 Home Page (Landing)

**Goal:** Create an immersive first impression with strong Ghibli vibes.

**Features:**

* Animated hero section with floating soot sprites
* Featured “Dish of the Day” 
* Background music toggle (Ghibli-inspired)
* Dynamic theme switch:

  * 🌞 Day mode
  * 🌙 Night mode


### 🍜 Menu Page (Main Attraction)

**Goal:** Showcases foods from various Ghibli movies.

**Includes foods from:**

* Spirited Away
* My Neighbor Totoro
* Princess Mononoke
* Ponyo
* Howl’s Moving Castle
* Kiki’s Delivery Service
* The Tale of the Princess Kaguya
* Whisper of the Heart
* The Cat Returns
* From Up on Poppy Hill

**Features:**

* Grid layout of dishes
* Filtering options:

  * By movie
  * By type (dessert, meal, snack)
* Search functionality
* Hover animations (steam effects)
* Flip cards to reveal descriptions

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/ghibli-bites.git

# Navigate into the project
cd ghibli-bites

# Install dependencies
npm install

# Run the development server
npm run dev
```
